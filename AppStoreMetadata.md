## Description (4000 chars max)

**把乱糟糟的下载文件夹，变成井井有条的归档系统——无需订阅，数据永远不离开你的 Mac。**

Tidy 是一款专为 Mac 用户设计的智能文件整理工具。它自动扫描你的下载文件夹、桌面和文稿，识别截图、PDF、安装包、重复文件，并给出清晰的整理建议——你确认一下，它立刻执行。

### 核心功能

**🤖 AI 智能分析**
接入 Claude API 后，Tidy 可以分析文件内容，理解文件是什么、属于哪里，生成精准的归档路径建议。没有 API Key 也能使用基础的规则引擎。

**📁 整理建议（Bundle）**
Tidy 每周自动把相关文件组合成「整理建议」——比如"本周截图 23 张→存到 Screenshots/2025-01"。你一键确认，文件自动归位。

**🔁 重复文件检测**
发现完全相同的文件，每组保留最新版本，其余全部移入废纸篓。支持一键清理或手动选择。

**🛡 隔离区（30 天缓冲）**
不确定要不要删的文件，先放进隔离区。30 天内随时一键恢复原位。过期后才真正删除，再也不用担心误删。

**⏪ 所有操作均可撤销**
每一步整理都记录在操作历史中，支持一键撤销。

**📋 版本文件管理**
自动发现 report_v1、report_v2、report_final 这类版本文件，帮你保留最新版、归档旧版。

**🔍 全局文件搜索**
在所有已授权目录中快速搜索文件，支持文件名和路径模糊匹配。

**📌 自定义规则**
确认一条整理建议后，Tidy 自动学习规则。下次同类文件出现时，直接按规则处理，无需再次确认。

**🔒 完全本地化**
Tidy 只访问你明确授权的文件夹，不上传任何文件或元数据，无需注册账号。AI 分析通过你自己的 API Key 直接调用，数据从不经过第三方服务器。

## What's New (4000 chars max)

### 版本 0.9.5

**全新隔离区体验**
- 隔离区现在清晰区分「活跃」和「已过期」文件
- 支持安全一键清理低风险过期项（dmg/pkg 和重复文件）
- 每个文件都有「显示」「恢复」按钮和右键快捷菜单

**整理建议更稳定**
- 修复了整理建议在确认后重新出现的 bug
- Bundle ID 现在使用周级别稳定 key，不再每天重新生成
- 已确认/已执行的建议不会被错误地重置为待处理

**防止扫描自身项目**
- 新增 git 仓库自动跳过：包含 .git 文件夹的目录不会被索引
- 自动跳过 node_modules、.build、DerivedData、Pods 等开发构建目录
- 支持在「偏好设置」中手动添加排除目录

**无障碍与键盘操作**
- 所有文件行支持 VoiceOver 组合朗读
- 主要操作按钮添加了语义化 accessibilityLabel
- 文件列表支持 Cmd+Delete 快捷键
- 超长列表支持分页（每页 50 条），避免大量文件时界面卡顿

**归档数据一致性**
- 每次刷新时自动修复「文件已在归档目录但数据库显示未归档」的不一致问题

## Privacy Policy Statement

Tidy 不收集任何个人数据，不上传文件内容，不注册账号，不连接 Tidy 服务器。

- 文件访问：仅限用户在设置中明确授权的文件夹
- AI 分析：通过用户提供的 API Key 直接调用 Anthropic API，文件名和摘要发送至 Anthropic 进行分析，遵循 [Anthropic 隐私政策](https://www.anthropic.com/privacy)
- 本地存储：所有整理记录、规则和统计数据保存在 `~/Library/Application Support/Tidy2/` 目录下
